Bodd

Touch screen interface for a 3D body scanner

Responsibilities

Interaction design, UX, UI, illustration, animation

Period

2022-2024

Background

Bodd have a bespoke piece of hardware which scans bodies for measurments and wellness insights. The following is a brief collection of experiences designed for the scanner's touch screen interface.

The self serve flow for how a user gets scanned. Users need to be completely self reliant on performing a scan, hence a clear instructional user experience was designed to allow them to complete a scan unsupervised.

The Bodd scanner also offered a hand scan function for sizing gloves. This experience guides users through the hand scanning process.
