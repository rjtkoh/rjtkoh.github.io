Les Verbes

UI and interaction design for a handy French learning app

Platforms

iOS, Android

Responsibilities

UI design, interaction design, branding

Period

2019

Background

Les Verbes is an iOS French language learning app created by Wave Digital. This React Native built app aimed to simply challenge users through daily quizzes prompting their knowledge of French verb conjugations. As it was an internal green fields project, it was free of the usual creative constraints of legacy design and code. I thoroughly enjoyed designing the UI through to the finishing touches on the micro-interactions.
